# Smart Leads Dashboard

A full-stack MERN Lead Management Dashboard built using React, TypeScript, Node.js, Express, MongoDB, and TailwindCSS.

---

# Features

## Authentication
- JWT Authentication
- User Registration & Login
- Protected Routes
- Password Hashing using bcrypt
- Role-Based Access Control

---

## Leads Management
- Create Lead
- Update Lead
- Delete Lead
- View Leads List
- View Single Lead Details

---

## Advanced Features
- Search by Name or Email
- Filter by Status
- Filter by Source
- Multiple Filters Together
- Sort by Latest & Oldest
- Backend Pagination
- CSV Export
- Debounced Search
- Dark Mode Support

---

# Tech Stack

## Frontend
- React.js
- TypeScript
- TailwindCSS
- Axios

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- TypeScript
